#+title: geojson-rgf93-wgs84

* Usage

With [[https://leiningen.org/][leiningen]] installed, clone the repository, then

: ~$ cd geojson-rgf93-wgs84
: ~$ lein run ~/my-input-file.geojson ~/my-output-file.geojson
